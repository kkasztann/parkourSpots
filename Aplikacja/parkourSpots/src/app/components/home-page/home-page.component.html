parkour SPOTS

Umożliwia utworzenie bazy miejsc do trenowania parkour, poprzez dodawanie nowych spotów, które następnie w łatwy sposów można zlokalizować na mapie. Dodane spoty są widoczne przez wszystkich w związku z czym baza może być tworzona i rozwijana przez wiele osób jednocześnie.

Kod źródłowy aplikacji można odnaleźć odwiedzając profil autora na platformie GitHub
